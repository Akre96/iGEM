CBCR Predictive Model

This is a new program that uses profile Hidden Markov Models and multiple sequence alignments to predict that the given cyanobacteriochrome (CBCR) will express blue, red, or yellow pigment. For more information on this model and the motivation please visit our teams wiki!


Input is one or more CBCR protein sequences in FASTA format. To test server functionality, please see 'Sample data' below. Note: [name] will only accept characters corresponding to the 20 proteinogenic amino acid residues -- other characters will result in an error message.
